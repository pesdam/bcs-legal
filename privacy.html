Privacy Policy

Application: BCSInvoices  |  Publisher: BCS Engineers, Inc.  |  Effective date: April 21, 2026

1. Scope

This Privacy Policy describes how the BCSInvoices application ("Application") handles data. The Application is an internal tool developed by BCS Engineers, Inc. ("BCS") for the exclusive purpose of automating invoice creation within BCS Engineers' own QuickBooks Online account. The Application is not offered to the public and does not process data of any organization other than BCS Engineers, Inc.

2. Data the Application Accesses

The Application accesses the following categories of data:

  • Internal invoice data from BCS Engineers' private Microsoft Azure SQL database, including invoice headers, line items, client names, project numbers, and billing amounts.
  • QuickBooks Online data belonging to BCS Engineers, Inc., including customer records, invoice records, and related accounting metadata, accessed via the official Intuit API under OAuth 2.0 authorization.

3. How Data Is Used

Data accessed by the Application is used solely to:

  • Retrieve billing information from BCS Engineers' internal systems.
  • Create, update, and reconcile invoice records in BCS Engineers' QuickBooks Online account.
  • Maintain an internal audit log of operations performed by the Application.

4. Data Sharing

The Application does not share, sell, rent, or disclose data to any third party. Data transmission occurs only between:

  • BCS Engineers' internal Azure SQL database,
  • The Application running on BCS Engineers' internal infrastructure, and
  • Intuit's QuickBooks Online API endpoints.

Intuit's handling of data transmitted through its API is governed by Intuit's own privacy policy, available at https://www.intuit.com/privacy/.

5. Data Storage and Security

The Application runs on infrastructure controlled by BCS Engineers, Inc. OAuth tokens and database credentials are stored in configuration files with restricted file-system permissions and are not transmitted outside of BCS Engineers' environment except as required to authenticate with the Intuit API. Communications with Intuit and Azure SQL occur over TLS-encrypted connections.

6. Data Retention

The Application does not maintain its own long-term data store. Any operational logs generated by the Application are retained solely within BCS Engineers' internal systems for troubleshooting and audit purposes, and are subject to BCS Engineers' general data retention practices.

7. No Personal Data of End Consumers

The Application does not process personal data of consumers or members of the general public. All data handled relates to business-to-business transactions between BCS Engineers, Inc. and its corporate clients.

8. User Rights

Because the Application is used exclusively by authorized personnel of BCS Engineers, Inc., any questions regarding data access, correction, or deletion should be addressed through BCS Engineers' internal IT and administrative channels.

9. Changes to This Policy

BCS may update this Privacy Policy from time to time. The effective date at the top of this document will reflect the most recent revision.
